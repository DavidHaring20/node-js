Getting started

What is Node.js ?

Node.js is open source, cross-platform, runtime environment that runs on V8 engine and can be executed outside of the web browser. It is based on JavaScript and we can say that it is server-side orientated.

How to start it ?

Firstly, we need an IDE. I chose Visual Studio as it is well optimized for Node.js. After we download it, we can open the IDE and head straight to the terminal. Firstly, we need to check if there is already Node.js installed on our PC:

$ node -v

If you don't have any version go here to download Node.js. Just choose LTS version and follow installation steps. Once it is downloaded, re-enter the the command from above. This is the wanted output:

$ v14.15.1

Once you got this output or some other version, congratulations, you did it.

JavaScript exercises

Exercise - Add numbers from string to float 
                const numberOne = "1.10";
                const numberTwo = "2.30";
                const result = parseFloat(numberOne) + parseFloat(numberTwo);

                console.log(result);

                // Expected output: 3.40
Exercise - Define an object 
                const myObject = {
                    "name" : "David",
                    "age" : 20  
                }
                console.log("I am " + myObject.name + " and I am " + myObject.age + " years old.");
                
                // Expected output: I am David and I am 20 years old.
Exercise - Add a property 
                const stackOverflow = {};
                stackOverflow.isAllowed = true;
                console.log(stackOverflow.isAllowed);

                // Expected output: true
Exercise - Remove a property 
                const thisSong = {
                    "description": "The best song in the world."
                }
                delete thisSong.description;
                console.log(thisSong.description);

                // Expected output: undefined
Exercise - Populate Array 
                const friends = [];
                friends.push("Joe");
                friends.push("Johny");
                friends.push("Jack");
                friends.push("Julius");
                console.log(friends);

                // Expected output: [ 'Joe', 'Johny', 'Jack', 'Julius' ]
Exercise - Get the index of the first occurence of some value 
                const significantMathNumbers = [0, 2.718, 3.14159, 1729];
                console.log(significantMathNumbers.indexOf(1729));

                // Expected output: 3
Exercise - For loop 
                const numbers  = [5, 3, 2, 7, 11, 12, 0, -20, 6];
                const discardedNumbers = [];

                for (let i = 0; i < numbers.length; i++) {
                    
                    let number = numbers[i];
                    
                    if (number < 0 || number > 6) {
                        console.log(number);
                    } else {
                        discardedNumbers.push(number)
                    }
                }
                console.log("Discarded numbers: " + discardedNumbers); 

                // Expected output: 7 11 12 -20
                                    Discarded numbers: 5,3,2,0,6

How do you start Node.js ?

Let's say that this is the look of our folder.

                /path/to/- dir0  
                        |- dir1
                        |- dir2 - myJsFile.js
                        |- dir3

If we would like to run our file(myJsFile.js), we firstly need to get into that directory.

cd /path/to/dir2

Now, we are in the same directory as our wanted .js file. Now we use command node <file_name>

node myJsFile.js
